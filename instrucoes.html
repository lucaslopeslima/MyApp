Desafio

Esse foi o meu primeiro projeto de C.R.U.D, e foi um baita desafio. Logo não poderia perder tempo com design tendo apenas um dia e meio, por isso utilizei um template que eu tinha. O template já tinha uma imagem de background, que eu troquei, e a barra lateral funcionando. Para o resto eu decidir não usar Bootstrap, pois faltava pouca coisa, era bem simples e seria um arquivo a menos para carregar.

CRUD

Loucura! Até então só tinha trabalhado com arrays(fiz um jogo de bingo), então sair de arrays e ir direto pro CRUD completo, que trabalha com objetos, foi uma loucura. Pesquisei bastante, vi tutoriais e tirei dúvidas em fóruns. O CRUD foi feito utilizando JS puro, e localstorage para persistência de dados. A entrada dos dados acontece tanto pela página de cadastro quanto pelo botao editar na pagina de banco de dados. A lista de cadastro é lida sempre que a página de banco de dados é carregada e sempre que um cadastro é editado ou deletado.
